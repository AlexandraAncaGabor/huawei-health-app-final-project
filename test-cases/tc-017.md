# **[TC-017] Nickname field validation**

### **Description**

A user is not able to set up a nickname containing symbols

### **Types of testing:**

- Functional testing,
- Negative testing.

### **Preconditions:**

- Huawei health app preinstalled on the phone,
- Available account,
- Huawei watch,
- App opened on device

|     | **Test steps**                                                     | **Expected results**                                                                         |
| --- | ------------------------------------------------------------------ | -------------------------------------------------------------------------------------------- |
| 1   | Access "Me" section                                                | Me section is selected                                                                       |
| 2   | Select the photo to redirect the user to the "Account center" area | "Account center" is selected                                                                 |
| 3   | Press "Nickname" button                                            | "Nickname" is selected                                                                       |
| 4   | Add symbols in the "Nickname" field      | Symbols are not allowed to be added in the field|
| 5   | Add only 1 character     | An error is displayed informing the user that the nickname must be at least 2 characters|
