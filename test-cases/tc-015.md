# **[TC-015] Phone field validation in "Personal info"**

### **Description**

Valid phone number can be added to the field

### **Types of testing:**

- Functional testing,
- Positive testing.

### **Preconditions:**

- Huawei health app preinstalled on the phone,
- Available account,
- Huawei watch,
- App opened on device.

|     | **Test steps**                                                     | **Expected results**                                                           |
| --: | ------------------------------------------------------------------ | ------------------------------------------------------------------------------ |
|   1 | Access "Me" section                                                | Me section is selected                                                         |
|   2 | Select the photo to redirect the user to the "Account center" area | "Account center" is selected                                                   |
|   3 | Choose "Personal info"                                             | "Personal info" is selected                                                    |
|   4 | Access "Phone number"                                 | "Phone number" is selected        |
|   5 | Add only few numbers (e.g. 0000)                      | Error message is shown       |
|   6 | Try to add letters in the field                       | Error message is shown       |
|   7 | Add a valid phone number                              | Valid phone number added.    |
