# **[TC-004] Watch face installation**

### **Description**

The watch face can be installed successfully and functions as expected.

### **Types of testing:**

- Functional testing,
- Positive testing.

### **Preconditions:**

- Huawei health app preinstalled on the phone,
- Available account,
- Huawei watch,
- App opened on device

|     | **Test steps**                            | **Expected results**                                |
| --: | ----------------------------------------- | --------------------------------------------------- |
|   1 | Go to "Device" section                    | "Device" is opened                                  |
|   2 | Press "Watch faces" button                | "Watch faces" is opened                             |
|   3 | Press install button on a free watch face | Face watch is installed. Check the download history |
